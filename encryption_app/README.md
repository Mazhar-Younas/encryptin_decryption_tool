# CryptoForge - Encryption Suite

**Master the Art of Cryptography**

A professional Streamlit web app supporting six cipher techniques:
Caesar, XOR, AES-256, RSA-2048, Monoalphabetic, and Playfair ciphers.

---

## Project Structure

```text
encryption_app/
|-- Home.py
|-- requirements.txt
|-- core/
|   |-- __init__.py
|   |-- ciphers.py
|   `-- ui_utils.py
`-- pages/
    |-- 1_Caesar_Cipher.py
    |-- 2_XOR_Cipher.py
    |-- 3_AES_Encryption.py
    |-- 4_RSA_Encryption.py
    |-- 5_Monoalphabetic_Cipher.py
    `-- 6_Playfair_Cipher.py
```

## Run the App

```bash
pip install -r requirements.txt
streamlit run Home.py
```
